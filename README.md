This GitHub repository contains Python programming exercises that are important for beginners. It helps new learners practice core concepts like strings, lists, dictionaries, and functions.

This repository contains Python practice exercises including:
•	Q1. Write a function that takes a string and returns:
-	reversed string
-	string with every 2nd character
•	Q2. Given a list of numbers, return a new list containing squares of even numbers only using list comprehension.
•	Q3. Write a function that counts frequency of each character in a string using a dictionary.
•	Q4. Write a function that takes a list and returns indices of all odd numbers using enumerate().
•	Q5. Write a function to rotate a list right by k positions using slicing.
•	Q6. Write a function that takes two lists and returns a dictionary using zip().
•	Q7. Using iter() and next(), manually iterate through a list and print elements one by one. Also, perform exception handling for the exception StopIteration using a try-except block.
•	Q8. Write a function that flattens a nested list using list comprehension.
•	Q9. Write a function that checks whether a string is a palindrome using slicing.
•	Q10. Write a function that returns the second largest number in a list (without sorting).
